# docker-influxdb-grafana-telegraf-rabbitmq

Start Influxdb Grafana Telegraf and RabbitMQ in Docker and starts graphing messages sent to amq.topic with routing key 'sensor/+/*'.

# Start

```
docker-compose up
```
